🎮 **GameHub**

⚠️ This project is currently under development. Features and games may change as the project evolves.

GameHub is a browser-based gaming hub built using HTML, CSS, and JavaScript.
It acts as a central platform where users can sign in, play mini-games, and compete on online leaderboards.

The platform integrates user authentication and stores scores using modern backend services.

---

🌐 **Live Demo**

[GameHub](https://linus9426.github.io/GameHub/)

---

🚀 **Features**

* User authentication using Firebase Authentication

* Online leaderboards powered by Supabase

* Multiple mini-games accessible from a single hub

* Interactive UI with dynamic JavaScript game logic

---

🎮 **Included Games**

– Memory Game
  
  * Card matching game with timer
  
  * Dynamic card generation
  
  * Sound effects and animations
  
  * Supabase leaderboard

– Spin the Wheel
  
  * Animated spinning wheel
  
  * Random category selection
  
  * Question system with timer
  
  * Score tracking and leaderboard
